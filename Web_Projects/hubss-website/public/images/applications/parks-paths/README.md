## Shot guidance

- Multi-use trail or park plaza with StreetBond colour coating or DuraShield sealcoat
- Green, natural surroundings contrast beautifully against a freshly coated path
- Cyclists, joggers, or families in the frame communicate recreational use
- Avoid winter shots — the "green and alive" aesthetic sells parks applications best
