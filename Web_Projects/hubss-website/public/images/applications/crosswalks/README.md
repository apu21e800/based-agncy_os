## Shot guidance

- The strongest shot in the HUB portfolio: colourful decorative crosswalk, aerial or elevated
- Pride crosswalks, Indigenous art crosswalks, or high-vis yellow/white thermoplastic all work
- Pedestrians crossing add life and scale — get consent or shoot from public street
- Canadian municipality context preferred
