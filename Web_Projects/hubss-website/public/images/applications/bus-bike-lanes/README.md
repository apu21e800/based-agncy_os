## Shot guidance

- Red MMA bus lane or green StreetBond bike lane along an urban corridor
- Long perspective shot communicates "Complete Streets" and scale
- Cyclists or a bus in the frame reinforces the use case
- Daytime, good light — colour saturation is the key selling point
