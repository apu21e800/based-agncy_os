## Shot guidance

- A gateway intersection or BIA corridor treatment that gives a neighbourhood a distinctive look
- Should feel like "placemaking" — distinctive design, not generic road markings
- Include surrounding context (storefronts, streetscaping) to show the full civic identity effect
- Elevated angle works well — shows the full pattern from above
