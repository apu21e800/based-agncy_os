## Shot guidance

- The most visually striking custom DecoMark installation in HUB's portfolio
- Aerial or elevated angle captures the full mural / artwork
- Indigenous art installations, Pride crosswalks, neighbourhood murals all work well
- People in the frame (with consent) communicate community and scale
- Colour accuracy is critical — this sells the "Pantone-matched" promise
