## Shot guidance

- Lead with the "wow" — a stunning brick or cobblestone pattern at an entrance plaza or gateway intersection
- Colour contrast is the hero: warm tones on dark asphalt
- Wide angle shows scale; close-up shows pattern quality — ideally include both
- A "before asphalt / after StreetPrint" pair is highly persuasive
