## Shot guidance

- Runway or taxiway markings — yellow hold-short lines, runway designation numbers, threshold markings
- Aerial or eye-level perspective along a runway communicates scale and precision
- Night shot with retroreflectivity visible is a powerful trust signal for safety-critical buyers
- Regional airport or private airstrip is fine — does not need to be a major airport
